<p align="center"><b>Overview</b></p>

<p>
<strong>SoulDrift</strong> is an AI/ML-based web application focused on mental well-being, habit formation, and daily productivity.
It leverages <strong>Natural Language Processing (NLP)</strong> and <strong>TensorFlow-based models</strong> to analyze user input,
track behavioral patterns, and provide meaningful analytics and encouragement.
</p>

<p>
The platform combines emotional awareness with structured productivity tools to help users build consistency,
reflect on their mental state, and stay motivated.
</p>

<hr/>

<p align="center"><b>Core Features</b></p>

<p><b>Journal-Based Mood Tracker</b></p>
<ul>
  <li>Users log daily moods through journal entries</li>
  <li>NLP models analyze text input to detect emotional patterns</li>
  <li>Visual analytics display mood trends over time</li>
  <li>Helps users understand emotional fluctuations and triggers</li>
</ul>

<p><b>Mood-Based Music Recommendation</b></p>
<ul>
  <li>Music suggestions adapt dynamically to the userâ€™s current mood</li>
  <li>Mood data from journal entries drives recommendations</li>
  <li>Designed to help users relax, focus, or uplift themselves</li>
</ul>

<p><b>Habit Tracker with Predictive Analytics</b></p>
<ul>
  <li>Users can create and track personal habits</li>
  <li>Displays streaks, consistency metrics, and completion rates</li>
  <li>AI-driven analytics estimate habit continuation likelihood</li>
  <li>Encourages users through personalized feedback</li>
</ul>

<p><b>Gamified Daily Task System</b></p>
<ul>
  <li>Users can add, manage, and complete daily tasks</li>
  <li>Tasks reward experience points (EXP)</li>
  <li>Level-based progression system encourages consistency</li>
  <li>Promotes productivity through gamification</li>
</ul>
